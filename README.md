# Description
  毎日の投資日記アプリになります。ホーム画面でその月の番号をクリックするとメッセージの入力フォームに飛べるようになっており、アイコンをクリックするとカレンダーを閲覧できます。入力したメッセージはマイページで閲覧できるようになります。
